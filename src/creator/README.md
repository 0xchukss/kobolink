# Creator

Creator listings are manual attachments of real X status posts. KoboLink does not scrape, auto-detect, fetch media, or publish the X post for the creator. Fans use the stored X URL to open the original post on X.

Creators provide:
- X handle
- X status URL
- Post content copied by the creator
- Optional media links
- Category
- Suggested tip in Naira/Kobo
- Arc wallet payout address

The public feed only includes creator-attached X status posts whose URL handle matches the creator handle.
